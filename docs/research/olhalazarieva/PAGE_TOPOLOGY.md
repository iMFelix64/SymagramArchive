# olhalazarieva.com topology

## Sections

1. Fixed header
   - Overlay chrome with blend-mode difference.
   - Logo left, bracket nav center, contact right.

2. Loader
   - Full viewport.
   - Percent counter fades out before full page interaction.

3. Hero
   - White background.
   - Giant condensed title.
   - Central image/card and small labels.
   - Bottom collaboration/recent-work links.

4. About / kinetic statement
   - Black background.
   - Large distorted uppercase text.
   - Sticky header remains readable.
   - Supporting images and monospaced paragraphs.

5. Works
   - Mostly dark.
   - Large media tiles.
   - Hover/reveal motion and split "view case" links.

6. Connect / footer
   - White background.
   - Sparse form lines and large contact typography.

## Implementation target in this repo

Implemented as `olha-effect.html`, with companion `olha-effect.css` and `olha-effect.js`, preserving the existing TwoDiskGarage archive page.
